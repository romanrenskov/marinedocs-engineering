# Lib

Папка для будущих сервисных модулей.

- `analytics.ts` - Яндекс.Метрика, события формы, клики Telegram/WhatsApp/телефон.
- `seo.ts` - title, description, OG, schema.org, sitemap.
- В будущем: отправка заявки на email, Telegram, Google Sheets или CRM.

Данных достаточно для стартовой архитектуры, но реальные ключи и счетчики сюда не класть.
