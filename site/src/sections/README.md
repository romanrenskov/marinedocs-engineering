# Sections

Смысловые секции MVP:

- `Hero` - оффер, CTA, география, основные услуги.
- `Problem` - боли клиента и задачи перед проверкой, регистрацией, эксплуатацией.
- `Solution` - инженерный подход и корректная документация.
- `Services` - калибровки, обмеры, кренование, взвешивание, остойчивость, эксплуатационная документация, 3D-геометрия.
- `Process` - 6 шагов работы от задачи до сопровождения замечаний.
- `Cases` - типовые задачи без фейковых отзывов.
- `Pricing` - индивидуальная оценка и факторы стоимости.
- `FAQ` - осторожные ответы без неподтвержденных юридических обещаний.
- `CTA` - форма заявки и контакты.
