# Components

Базовые компоненты для будущего разнесения текущего `App.jsx`:

- `Header.tsx` - фиксированная навигация, бренд, Telegram, заявка, мобильное меню.
- `Footer.tsx` - краткое описание, политика конфиденциальности.
- `Button.tsx` - CTA и вторичные кнопки.
- `Container.tsx` - ограничение ширины и сетка.
- `Section.tsx` - общая структура секции и заголовка.

Данные уже достаточно понятны из текущего лендинга.
