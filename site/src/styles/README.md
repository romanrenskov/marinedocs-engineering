# Styles

Текущий стиль: dark navy background, cyan accents, белая типографика, техническая сетка, строгие карточки, чистый B2B-интерфейс.

Ограничения: без дешевого люкса, кислотных акцентов, перегруза анимациями, мультяшности и фейковых отзывов.

Будущая структура:

- `globals.css` - базовые стили и reset.
- `variables.css` - цвета, радиусы, тени, сетки, типографика.
- `utilities.css` - небольшие служебные классы.
