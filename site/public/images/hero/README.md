# Hero

Текущий визуальный ориентир: темный морской B2B-кадр с судном, портом и инженерными схемами. В проекте уже есть `public/images/marine-engineering-hero.png`.

Hero должен сразу считываться как морская инженерия и судовая документация, а не как стартап про ИИ, строительство или абстрактная технология.
