# SiteShipAI Business Brain

Эта папка - второй мозг проекта MarineDocs Engineering: продукт, аудитория, экономика, маркетинг, контент, дизайн, цели и будущие интеграции.

Перед работой открывай сначала `INDEX.md`, затем только нужный раздел. Не загружай всю папку целиком без необходимости.

## Текущий контекст

- Продукт: B2B-услуги по судовой инженерной документации, расчетам остойчивости, калибровочным таблицам, обмерам, кренованию, взвешиванию и сопровождению судовладельцев.
- Рабочее название: MarineDocs Engineering.
- География: Санкт-Петербург, Северо-Запад, Россия, возможны выезды на судно.
- Основное действие: получить консультацию или рассчитать стоимость работ.
- Тон: строгий инженерный B2B без агрессивного маркетинга и неподтвержденных обещаний.

## Что уже есть в проекте

- Лендинг на Vite + React.
