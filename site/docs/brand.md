# Brand

## Ощущение

Строгий инженерный B2B: надежность, точность, судовая специфика, документация, контроль, профессиональная коммуникация.

## Визуальный стиль

- Темно-синий / navy фон.
- Глубокий синий.
- Cyan / бирюзовые акценты.
- Белая типографика.
- Серые технические элементы.
- Чистая сетка, технические линии, аккуратные карточки услуг.

## Визуалы

Подходят: коммерческое судно, танкер, порт, инженер за чертежами, технические схемы, калибровочные таблицы, 3D-модель судна, интерфейс расчетной программы, судовая документация.

Не подходят: мультяшные суда, случайные стоковые люди, киберпанк, космический корабль, псевдотекст, агрессивная реклама, много анимаций ради анимаций.
