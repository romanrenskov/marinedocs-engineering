# Brief

## Основа

MVP-сайт для B2B-бизнеса в сфере морской инженерии и эксплуатационной документации для флота.

Рабочее название: MarineDocs Engineering.

## Что продаем

- Разработка эксплуатационной документации для судов.
- Калибровочные таблицы танков и цистерн.
- Обмеры цистерн и судовых конструкций.
- Кренование.
- Взвешивание.
- Расчеты остойчивости.
- Подготовка технической документации для сюрвейера, классификационного общества или регистра.
- Сопровождение судовладельца по вопросам документации.
- В перспективе: 3D-обмеры, 3D-модели, расчеты прочности, прибор контроля остойчивости.

## Для кого

Судовладельцы, судоходные компании, операторы флота, технические директора, суперинтенданты, капитаны, проектные организации, владельцы танкеров, сухогрузов, рабочих, коммерческих и маломерных судов.

## География

Россия, в первую очередь Санкт-Петербург, Ленинградская область, Северо-Запад и порты РФ. Возможны командировки по России. Часть работ можно делать удаленно по исходным данным.

## Главное действие

Оставить заявку на консультацию или расчет стоимости. Дополнительно: Telegram, WhatsApp, звонок, отправка ТЗ или документов на предварительную оценку.

## Ограничения

Не писать без подтверждения: `аккредитованы РМРС`, `официально одобрено регистром`, `100% примут документы`, `сертифицированная компания`.

Корректная формулировка: готовим техническую документацию для судовладельцев и сопровождаем процесс работы с замечаниями в рамках согласованного объема.
